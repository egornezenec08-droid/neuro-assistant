# 🧠 Neuro — Голосовой ассистент с искусственным интеллектом

**Neuro** — это современный веб-голосовой ассистент с русскоязычным интерфейсом, который работает прямо в браузере. Он понимает голос, отвечает на вопросы, показывает погоду, переводит текст, ищет информацию в Википедии и многое другое. Проект не требует серверной части и полностью работает на JavaScript (Frontend).

---

## ✨ Демонстрация возможностей

| Команда | Действие |
| :--- | :--- |
| 🎤 "Скажи погоду" | Показывает текущую погоду (с геолокацией или по городу) |
| 🗺️ "Открой карту" | Интерактивная карта на основе OpenStreetMap |
| 🔢 "Реши 2x + 5 = 15" | Решение уравнений и математических выражений |
| 📖 "Кто такой Пушкин?" | Поиск информации в Википедии |
| 🔄 "Переведи 'Hello' на русский" | Мгновенный перевод текста |
| ⏲️ "Таймер на 5 минут" | Обратный отсчёт с уведомлением |

---

## 🚀 Основные возможности

### 🧠 Интеллектуальный движок
- **Трёхуровневая система ответов**:
  1. Локальная база знаний (200+ городов, факты, рецепты)
  2. Wikipedia API (поиск фактологических данных)
  3. Бесплатные LLM (Gemini, Pollinations, DeepInfra) без ключей

### 🎤 Голосовое управление
- Распознавание русской речи (Web Speech API)

### 🌐 Встроенные сервисы (без API-ключей)
- Погода: Open-Meteo + wttr.in
- Карты: OpenLayers + OSM
- Курсы валют и криптовалют

### 🎨 Интерфейс
- **Адаптивный дизайн** (ПК, планшеты, телефоны)
- **Тёмная/светлая тема** (сохраняется в `localStorage`)
- Регулируемое положение «глаз» ассистента
- Окно чата с историей и кнопкой очистки
